# CamAtlas Ethics & Responsible Use Policy

CamAtlas is an open-source project designed to index and organize
webcam feeds that are intentionally published for public viewing.

This project is built with respect for privacy, legality, and
responsible data use.

## What CamAtlas Does

- Indexes webcams that are explicitly published on public websites
  (e.g. traffic cameras, weather cams, tourism webcams, wildlife cams)
- Collects basic metadata such as location, camera type, and source page
- Provides an open and transparent index for educational and informational use

## What CamAtlas Does NOT Do

- Does NOT scan IP address ranges
- Does NOT probe or discover private devices
- Does NOT attempt to access unsecured or private CCTV systems
- Does NOT bypass authentication, passwords, or access controls
- Does NOT collect or store personal data

## Data Collection Principles

- Only publicly available web pages are indexed
- Site terms of service and robots.txt are respected
- Requests are rate-limited to avoid server overload
- Camera owners may request removal at any time

## Legal & Ethical Commitment

CamAtlas is not a surveillance tool.
It does not monitor individuals or private property.
The project exists to document and organize publicly published
camera feeds in an ethical and responsible manner.

If you believe a camera listed by CamAtlas should not be included,
please open an issue or contact the maintainers for removal.

## Contributor Responsibility

All contributors are expected to follow this ethics policy.
Any code or contribution that violates these principles will
not be accepted into the project. Any contributers must disclose
the use of AI in thier contributions. Adquate comments must be 
added to any code to be accepted.
