# CamAtlas Sources Template

**IMPORTANT CONTRIBUTOR NOTES:**

- This file is a **template** for adding publicly accessible webcam sources.
- Add only webcams that are **intentionally public** â€” do NOT add private or restricted feeds.
- Each source entry should include:
  - **name**: descriptive camera name
  - **url**: link to the public webcam page
  - **type**: category (e.g., traffic, tourism, wildlife, citycams)
  - **country**: ISO country code or 2-letter abbreviation
- Keep the format consistent for easy automated parsing.
- Contributors: Add new entries **below existing examples**.
- Use this file as a reference before editing `sources.json`.
